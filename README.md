## Projeto Eleicões

Aplicação web para criação de eleições para cargos cadastrados, com geração de relatórios.

[DEMO](https://eleicoes.tec.pet/)

### Back-end API REST

- Spring Boot v2.3.0
    - Spring Web;
    - Spring Data JPA;
    - PostgreSQL Driver;
    - Spring REST Docs;
- Database Postgres;

### Front-end

- React v16.13.1 com TypeScript
- UI Componets via [Ant Design](https://ant.design/) 4.2

### Container
- Docker
